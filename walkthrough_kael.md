# Bienvenue sur le projet Velto Travel 🇨🇦

Salut Kael ! Ce dépôt contient l'ensemble de l'écosystème **Velto**, la plateforme de voyage intelligente spécialisée pour le marché canadien.

## 📁 Structure du Projet

Pour plus de simplicité, nous avons regroupé les trois piliers du projet dans ce dépôt unique :

1.  **`/site` (Landing Page)** : Le site principal construit avec **React + Vite + Framer Motion**. C'est ici que se trouve l'expérience de marque "Ailleurs. Pour beaucoup moins".
2.  **`/app` (Interactive Discovery)** : L'interface "Reels" immersive. Elle permet de défiler les deals voyage comme sur Instagram/TikTok, avec l'intégration d'**Éléonore AI**.
3.  **`/framer` (Composants)** : Une collection de composants React haute performance conçus pour être importés directement dans **Framer** (Hero, Bento Grid, Expert AI).

## 🚀 Comment lancer le projet localement

### Le Site Principal
```bash
cd site
npm install
npm run dev
```
Accédez ensuite à `http://localhost:5173`.

### L'App Discovery
Il suffit d'ouvrir le fichier `app/index.html` directement dans votre navigateur ou via un serveur local.
